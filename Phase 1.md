
# Phase 1 – Basic Tetris Logic (Text-Based)

## Goal
Create a very simple, playable version of Tetris to validate core mechanics.

## Scope
- No graphics
- No animations
- No real-time input

## Functional Requirements
- Represent a game grid (e.g. 10x20)
- Generate basic Tetris shapes
- Allow shapes to "fall" step by step
- Detect when a shape hits the bottom
- Stack shapes on the grid

## Non-Goals
- No scoring
- No rotation
- No line clearing
- No UI

## Audience
Non-technical users. Code must be explained in plain English.

## Success Criteria
- The concept of Tetris is recognisable
- Logic is readable and commented
- Game loop concept is clear
