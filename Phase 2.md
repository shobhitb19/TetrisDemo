
# Phase 2 – Core Tetris Rules

## Goal
Add the essential rules that make Tetris feel real and trustworthy.

## Functional Requirements
- Support multiple Tetromino shapes
- Allow rotation of shapes
- Detect filled horizontal lines
- Clear completed lines
- Shift remaining blocks downward

## User Interaction
- Basic keyboard controls (left, right, rotate, drop)

## System Behaviour
- End game when blocks reach the top
- Maintain a game loop

## Constraints
- Keep logic simple and readable
- Avoid performance optimisations

## Success Criteria
- Game behaves as expected
- No obvious rule-breaking
- Players trust the system
